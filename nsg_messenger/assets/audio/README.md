# Ringback-ассеты исходящего звонка

Синтетические тоны обратного сигнала («гудки») для КАЛЛЕРА в исходящем
звонке — проигрываются `call_ringback_player.dart` из
`call_overlay_host.dart` по стадиям `CallOutgoingRinging`.

| файл | стадия | что означает | тон |
|------|--------|--------------|-----|
| `ringback_connecting.wav` | 1 — `reachedPeer=false` | «дозвон до сервера» (invite ещё не доставлен) | двойной блип 520 Гц, цикл 1.0с |
| `ringback_ringing.wav` | 2 — `reachedPeer=true` | «звонит на устройстве» (invite доставлен) | гудок 425 Гц, 1с/3с, цикл 4.0с |

Формат: 8 кГц / mono / 16-bit PCM WAV. Тишина «вшита» в конец цикла —
`LoopMode.one` петляет бесшовно (стык в тишине, без щелчка).

Регенерация / подстройка (частоты, длительности, громкость):
```
python generate_ringback.py
```
Живая проверка звука — за постановщиком (в т.ч. iOS), см. задачу.
